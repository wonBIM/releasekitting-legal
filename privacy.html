ReleaseKitting — Privacy Policy

Last updated: 14 July 2026

What ReleaseKitting is

ReleaseKitting is a personal, self-hosted tool that generates short-form promotional videos from an app's own screenshots and uploads them to the operator's own social media accounts (YouTube, TikTok, Instagram).

It is not a public service. It runs locally on the operator's machine and is used by a single person — the operator — on their own accounts. There is no sign-up and there are no other users.

What data is collected

None from any third party. ReleaseKitting has no users other than its operator, and performs no analytics or tracking.

The following is stored locally on the operator's own machine and is never transmitted anywhere except to the platform APIs described below:

DataWhyWhere it is stored
OAuth access / refresh tokens for the operator's own YouTube, TikTok and Instagram accountsTo upload videos to those accountsA local .env file on the operator's computer
Screenshots and README text supplied by the operatorTo generate the promo video and its scriptLocal disk
Generated videos, scripts and captionsThe output of the toolLocal disk

What data is shared

A video file and its title, caption and hashtags are sent to the platform whose account the operator is uploading to — YouTube, TikTok, or Instagram — using that platform's official API, and only when the operator explicitly triggers an upload.

Screenshot images and README text are sent to Google's Gemini API in order to generate the video script and narration.

Nothing is sold, rented, or shared with anyone else.

Third-party data

ReleaseKitting does not read, collect, store, or process any other TikTok user's data. It does not access the operator's followers, viewers, comments, or any public content belonging to other people.

Retention and deletion

All data lives on the operator's own machine and can be deleted at any time by deleting the local files. Access tokens can be revoked at any time in the respective platform's account settings:

  • TikTok: Settings → Security & permissions → Manage app permissions
  • YouTube / Google: myaccount.google.com/permissions
  • Instagram / Meta: Settings → Apps and websites

Children

The tool is not directed at children and collects no data from anyone other than its operator.
